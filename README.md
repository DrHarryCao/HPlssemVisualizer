## 🌟 Introduction / 简介

**[EN]** This comprehensive suite of professional-grade tools empowers researchers to transform abstract PLS-SEM data—including validity matrices, path coefficients, and moderation effects—into publication-ready, high-resolution visualizations that meet the rigorous aesthetic standards of top-tier academic journals.

**[ZH]** 这是一套专为 PLS-SEM 分析打造的高端可视化工具集，能够将抽象的信效度矩阵、路径假设、调节效应等复杂模型数据，一键转化为符合学术顶刊审美标准的高清出版级图表。

## 🚀 Core Modules / 核心模块

### 1. Discriminant Validity Matrix Engine / 判别效度矩阵引擎
- **[EN]**: Automatically combines HTMT and Fornell-Larcker Criterion (FLC) data into a single creative matrix. Supports multiple geometries (hexagons, 2.5D spheres, soft squircles) and journal-specific color themes (Nature, Science, MISQ).
- **[ZH]**: 自动将 HTMT 和 Fornell-Larcker 准则 (FLC) 数据合并为单一创意矩阵。支持多种几何形态（六边形、2.5D悬浮球体、平滑圆角）以及特定期刊的配色主题（Nature、Science、MISQ）。

### 2. Reliability & Validity Rings / 信效度环形罗盘
- **[EN]**: Visualizes AVE, Composite Reliability (Rho_C), Rho_A, and Cronbach's Alpha in an intuitive concentric ring dashboard. Highlights failing metrics automatically.
- **[ZH]**: 以直观的同心环仪表盘可视化 AVE、组合信度 (Rho_C)、Rho_A 和 Cronbach's Alpha。自动高亮显示未达标的指标，提供出版级高分辨输出。

### 3. Path Analysis Decision System / 路径系数验证系统
- **[EN]**: Generates horizontal bar charts for structural model paths, mapping beta values with significance levels (p-values, T-values) using a clear, color-coded legend.
- **[ZH]**: 为结构模型假设路径生成水平条形图，使用清晰的颜色编码图例将 Beta 值与显著性水平（p 值、T 值）进行科学映射，支持自适应布局。

### 4. Moderation Effects Plotter / 调节效应斜率图
- **[EN]**: Plots interaction effects (+1 SD / -1 SD) with fully customizable typography and journal-specific contrast themes.
- **[ZH]**: 一键绘制调节交互效应斜率（+1 SD / -1 SD），提供完全可自定义的排版控制和特定期刊的高对比度学术主题。

### 5. Demographics & Spatial Mapping / 人口统计与空间分布映射
- **[EN]**: Transforms sample demographic data into elegant SSCI-style charts and interactive spatial distribution maps using dynamic matching algorithms.
- **[ZH]**: 通过动态匹配算法，将样本人口统计数据转化为优雅的 SSCI 顶刊风格图表，并生成支持冷暖色调的交互式空间人口分布地图。

## 📥 Installation & Usage / 安装与使用

**[EN]**
1. Ensure Python 3.8+ is installed.
2. Install dependencies: `pip install customtkinter pandas numpy matplotlib pillow openpyxl`
3. Run the respective `.py` module to launch the GUI interface.

**[ZH]**
1. 确保已安装 Python 3.8 或更高版本。
2. 安装运行依赖：`pip install customtkinter pandas numpy matplotlib pillow openpyxl`
3. 运行相应的 `.py` 脚本模块即可启动可视化图形界面。
